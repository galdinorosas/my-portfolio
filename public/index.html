About Me

Hello! My name is Dino Rosas, I'm a front-end web developer from the Bay Area, CA. I was always curious and wanted to understand how the web works and as a result I got into into web development. The idea of starting from a blank canvas/web-page is one of the reasons why I enjoy coding, then to able to "paint" on this canvas by using HTML/CSS, afterwards Javascript is used to add in any neat features to bring the canvas/web-page alive.

I received my bachelors degree in Chemical Engineering from San Jose State University and I currently work as a junior reliability engineer in Milpitas, CA. I first was introduced to programming in college with an intro to java programming course and I loved it! After I graduated and started working, I had some free time on my hands so I picked up programming as a hobby. I enjoyed learning about how websites are created from general courses on Udacity and Code-academy but I thought the material was not very in-depth. Realizing at how much time I was spending and how easy it was to get into rhythm when coding, I decided to pursue web development as a career. I have since then enrolled in courses at Thinkful to improve and learn more about front-end web development.
